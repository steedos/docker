# Steedos Docker

## Steedos Community 

```shell
cd steedos-community
docker-compose up

# 使用国内镜像
docker-compose -f docker-compose-cn.yml up
```

Steedos Community will running at http://hostname:3000

## Steedos Enterprise 

```shell
cd steedos-enterprise
docker-compose up

# 使用国内镜像
docker-compose -f docker-compose-cn.yml up
```

Steedos Enterprise will running at http://hostname:3000
